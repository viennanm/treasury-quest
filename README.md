# Treasury Quest – Interactive CFO Challenge

Treasury Quest is an interactive browser-based treasury simulation game.

The player takes the role of a corporate treasurer or CFO and has to manage liquidity, risk, forecast accuracy, bank relationships and governance through realistic finance and treasury events.

The game is designed for:

* CFOs
* Treasurers
* Finance managers
* Controllers
* Entrepreneurs
* Students of finance and economics

## Purpose of the Game

The goal is to make key treasury principles understandable through interactive decisions.

Players experience how treasury decisions affect:

* Cash position
* Risk exposure
* Forecast accuracy
* Bank relationship
* Governance score
* Final treasury maturity level

The game is not a financial model and not investment advice. It is an educational simulation designed to explain economic and treasury trade-offs.

## Game Mechanics

The game contains 40 realistic treasury event cards.

Each event card presents:

1. A realistic finance or treasury situation
2. Three decision options
3. Direct effects on five key indicators:

   * Cash
   * Risk
   * Forecast Accuracy
   * Bank Relationship
   * Governance Score

At the end of the game, the player receives a Treasury Health Score and a final maturity level.

## Available Game Modes

The game includes two modes:

* 12-card sprint
* 40-card full game

The language can be switched between German and English.

## Treasury Levels

At the end of the game, the player reaches one of five levels:

1. Liquidity Firefighter
2. Cash Controller
3. Treasury Manager
4. Senior Treasurer
5. Global Treasury Architect

## How to Play

Open the game in a browser and choose the language and game mode.

Then select one of the three available decisions for each event card. The dashboard updates automatically after every decision.

## Technical Information

This game is built as a standalone HTML file using:

* HTML
* CSS
* JavaScript
* Canvas-based charts

No backend, database or external framework is required.

## Possible Use Cases

Treasury Quest can be used as:

* A website lead-generation tool
* A finance education game
* A workshop introduction
* A treasury training exercise
* A demonstration of AI-supported finance process thinking

## Disclaimer

This game is a simplified educational simulation. It does not represent professional financial, legal, tax or investment advice.
